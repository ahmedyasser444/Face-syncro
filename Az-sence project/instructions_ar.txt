
# تعليمات تشغيل المشروع

1. تثبيت المتطلبات:
   أولاً، قم بتثبيت جميع المكتبات المطلوبة باستخدام هذا الأمر:
   ```bash
   pip install -r requirements.txt
   ```

2. تشغيل البرنامج:
   بعد تثبيت المتطلبات، يمكنك تشغيل البرنامج باستخدام هذا الأمر:
   ```bash
   python face_syncro2,0.py
   ```

3. سيتم فتح نافذة الكاميرا، حيث سيبدأ النظام في التعرف على الوجوه وتسجيل الحضور.
   يتم تسجيل الحضور يدويًا بعد التعرف عبر زر في الواجهة.
